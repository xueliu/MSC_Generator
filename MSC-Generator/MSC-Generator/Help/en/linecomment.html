LINECOMMENT - Diagram Element

Using the command linecomment a line diagram comment (without a comment box) can be added. The comment is attached to the lifeline of a process or an actor. To determine the position of the comment, use the parameters instanceid and orientation The comment text is specified by the parameter text. Optionally, the horizontal line that connects the comment to its corresponding lifeline can be hidden by setting the parameter format.
Syntax
linecomment: instanceid, text, orientation, format
Parameters
instanceid
The ID of the lifeline at which the comment should be displayed.
text
The text that should be displayed as a comment. If your text contains punctuation, you must put the complete text in quotes ("").
orientation (optional)
Sets the position of the comment relative to the lifeline selected by the parameter instanceid.
Value Description
l The comment will be displayed to the left of the lifeline
r The comment will be displayed to the right of the lifeline
The default value is for this parameter is "l".
format (optional)
Determines whether the comment should be connected to its corresponding lifeline by a horizontal line.
Value Description
* The comment will be displayed without the horizontal line
- The comment will be displayed with the horizontal line
The default value for this parameter is "-".
Example
DiagramName: Comments
DiagramStyle: uml
PageSize: auto
PageMargins: 10,10,10,10
left: 100
right: 50

process: p1, Process 1
process: p2, Process 2

comment: p1, Comment 1, r;
comment: p2, Comment 2, l;
comment: ENV_RIGHT, Comment 3
comment: ENV_LEFT, Comment 4;
linecomment: p2, Comment 5;
linecomment: p1, Comment 6, r, *;
commentoverall: Comment 7;
