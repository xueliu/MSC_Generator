MEASUREEND - Diagram Element

Use the commands measurebegin and measureend to model a measurement. The command measureend defines the end of the measurement. The beginning and the end of a measurement are connected with a line on the generated diagram. It is not possible to nest multiple measurements on the same lifeline. For this purpose use the commands measurestart und measurestop which are displayed without a line connecting the start and the end.
Syntax
measurebegin: instanceid, text
Parameters
instanceid
The ID of the lifeline on which the measurement should be ended.
text (optional)
The text that should be displayed at the end of the measurement. If your text contains punctuation, you must put the complete text in quotes ("").
Examples
DiagramName: Measurements
DiagramStyle: uml
PageSize: 550,auto
PageMargins: 10,10,10,10
Left: 50
Right: 50

process: p1, "Process 1"
process: p2, "Process 2"

measurebegin: p1, M1
measurebegin: p2, M2, r, *
msg: p1, p2, Message 1;
;
;
msg: p2, p1, Message 2
measureend: p1
measureend: p2, end;
