BACKCOLOR - Formatting

Using the command backcolor it is possible to set the background color of the following diagram elements. The text background color is specified in the parameter color and is valid until the next occurrence of the command backcolor. The text background color is only applied to diagram elements which are not enclosed in a box or rectangle. For such elements the command fillcolor should be used instead.
Syntax
backcolor: color
Parameters
color
The color which should be used for the background of text labels. The color must be given in HTML format, for example, "red", "darkgreen", "#FF22A0". If "none" is specified then the standard value for "white" will be used.
Example
PrintAuthor: yes
PrintCompany: yes
PrintVersion: yes
PrintDate: yes
PrintCreationDate: yes
PrintFileName: yes
Font: "Arial", "10", "Regular"
DiagramName: Colors
DiagramStyle: uml
PageMargins: 5, 5, 5, 5
PrintFootLine: yes
Author: Martin Kotowicz
Company: ITESYS GmbH
Version: 1.0
Date: 20.10.2006
PageSize: auto, auto

process: P1, "Process 1"
process: P2, "Process 2"
process: P3, "Process 3"

backcolor: yellow
fillcolor: #A0A0FF
msg: P1,P2, backcolor
state: P3, fillcolor;

backcolor: none
fillcolor: none
textcolor: red

state: P2, textcolor;
