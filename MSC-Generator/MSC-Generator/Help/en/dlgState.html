Dialog State

In this dialog you can change the properties of the diagram element State. The changes will take effect in the diagram only after you click the button "Apply" or "Apply and Close". The button "Cancel" closes the dialog and you will lose all changes which have not been applied.
State Text
The text that will be displayed as state invariant.
See also
state stateoverall
Format Standard
The state invariant will be displayed in a state symbol.
See also
state stateoverall
Format Alternative
The state invariant will be displayed in curly brackets. This option is not available for the command stateoverall.
See also
state
Alignment Top
The state invariant will be displayed at the top of the diagram line.
See also
state stateoverall
Alignmnet Bottom
The state invariants will be displayed at the bottom of the diagram line.
See also
state stateoverall
Cancel
Closes the properties dialog without applying any changes to the diagram.
Apply
Applies all changes to the diagram.
Apply and Close
Applies all changes to the diagram and closes the properties dialog.
