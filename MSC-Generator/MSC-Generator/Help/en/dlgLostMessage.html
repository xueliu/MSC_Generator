Dialog LostMessage

In this dialog you can change the properties of the diagram element Lost Message. The changes will take effect in the diagram only after you click the button "Apply" or "Apply and Close". The button "Cancel" closes the dialog and you will lose all changes which have not been applied.
Message Text
The text that should appear as the lost message.
See also
lost
Gate Text
The text that should be displayed below the gate of the lost message.
See also
lost
Alignment Left
The lost message will be displayed to the left of the lifeline.
See also
lost
Alignment Right
The lost message will be displayed to the right of the lifeline.
See also
lost
Cancel
Closes the properties dialog without applying any changes to the diagram.
Apply
Applies all changes to the diagram.
Apply and Close
Applies all changes to the diagram and closes the properties dialog.
