PAGESIZE - Formatting

Use the command pagesize to set the size of the diagram. The command can be called with two sets of parameters, depending of the values you specify. It is possible to specify a page size from A0 to A5 by setting the parameter size. The orientation could be set in the parameter orientation. Alternatively, it is possible to set the height and width of the diagram using the parameter width and height. The optional parameter unit to specify the units used.
Syntax
pagesize: size, orientation
or
pagesize: width, height, unit
Parameters
size
Value Description
A0 The diagram has the size of a DIN A0 page (118,9cm x 84,1cm)
A1 The diagram has the size of a DIN A1 page (84,1cm x 59,4cm)
A2 The diagram has the size of a DIN A2 page (59,4cm x 42cm)
A3 The diagram has the size of a DIN A3 page (42cm x 29,7cm)
A4 The diagram has the size of a DIN A4 page (29,7cm x 21cm)
A5 The diagram has the size of a DIN A5 page (21cm x 14,8cm)
orientation (optional)
Value Description
h Landscape
v Portrait
The default value for this parameter is "h".
width
This parameter sets the width of the diagram. The unit of the value is specified by the parameter unit. Decimal numbers should be entered in quotation marks.
If the value "auto" is specified then the width is calculated automatically depending on the number of the defined processes.
height
This parameter sets the height of the diagram. The unit of the value is specified by the parameter unit. Decimal numbers should be entered in quotation marks.
If the value "auto" is specified then the height is calculated automatically depending on the number of diagram lines.
unit (optional)
This parameter defines the units used for the values of the parameters width and height. This parameter is ignored if the value "auto" is specified for width and height.
Value Description
pixel Pixel
mm Millimeter
cm Centimeter
inch Inch
The default value for this parameter is "pixel".
