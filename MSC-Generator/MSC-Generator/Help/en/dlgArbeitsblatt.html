Dialog Workspace

In this dialog you can change the properties of the diagram workspace. The changes will take effect in the diagram only after you click the button "Apply" or "Apply and Close". The button "Cancel" closes the dialog and you will lose all changes which have not been applied.
Page Size
Sets the size of the diagram sheet.
See also
pagesize
Units
Selects the units for the diagram worksheet size.
See also
pagesize
Width
Configures the width of the worksheet (including page margins).
See also
pagesize
auto Width
Configures the generator to automatically determine the width of the diagram based on the number of processes.
See also
pagesize
Height
Configures the height of the worksheet (including page margins).
See also
pagesize
auto Height
Configures the generator to automatically determine the width of the diagram based on the number of actions.
See also
pagesize
Margins Left
Configures the width of the left diagram margin.
See also
pagemargins left
Margins Top
Configures the height of the top diagram margin.
See also
pagemargins
Margins Right
Configures the width of the right diagram margin.
See also
pagemargins right
Margins Bottom
Configures the height of the bottom diagram margin.
See also
pagemargins
Orientation Landscape
Configures a horizontal paper orientation.
See also
pagesize
Orientation Portrait
Configures a vertical paper orientation.
See also
pagesize
Cancel
Closes the properties dialog without applying any changes to the diagram.
Apply
Applies all changes to the diagram.
Apply and Close
Applies all changes to the diagram and closes the properties dialog.
