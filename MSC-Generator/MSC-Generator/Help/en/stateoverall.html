STATEOVERALL - Diagram Element

Use the command stateoverall to define a state invariant spanning over all diagram processes. If you to model a state invariant only for a single or multiple instances, then use the command state instead.
Syntax
stateoverall: text, orientation
Parameters
text
The text that should be displayed as the state invariant. If your text contains punctuation, you must put the complete text in quotes ("").
orientation (optional)
Determines the vertical alignment of the state invariant in the diagram line.
Value Description
t The state invariant is top aligned
b The state invariant is bottom aligned
The default value for this parameter is "t".
Example
DiagramName: States
DiagramStyle: uml
PageSize: auto
PageMargins: 10,10,10,10
left: 100
right: 50

process: p1, Process 1
process: p2, Process 2
process: p3, Process 3

stateoverall: State 1;
state: p1, State 2,,t
state: p2, State\n3
state: p3, State 4,-,b;

state: p3, State 5,*,b
state: "p1,p2", State 6;
state: "p1,p3", State 7;
