Dialog TimeOut

In this dialog you can change the properties of the diagram element Timeout. The changes will take effect in the diagram only after you click the button "Apply" or "Apply and Close". The button "Cancel" closes the dialog and you will lose all changes which have not been applied.
Timeout Text
The text that should appear above the timer end point.
See also
timeout
Alignment Left
The timer end point will be displayed on the left of the lifeline.
See also
timeout
Alignment Right
The timer end point will be displayed on the right of the lifeline.
See also
timeout
Cancel
Closes the properties dialog without applying any changes to the diagram.
Apply
Applies all changes to the diagram.
Apply and Close
Applies all changes to the diagram and closes the properties dialog.
