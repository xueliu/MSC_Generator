CREATE - Diagram Element

Use the command create to send a create message to a process. The lifeline of the process must be reserved in advance using the command dummyprocess.
Syntax
create: sourceid, destinationid, message, name, description
Parameters
sourceid
The lifeline ID from which the create message is sent.
destinationid
The lifeline ID, which has already been reserved using the command dummyprocess, for which a process should be created.
message
The text that should appear as the create message text. If your text contains punctuation, you must put the complete text in quotes ("").
name
The name of the process. Some text that defines the process's name. The name is displayed in the diagram right below the process symbol. If your text contains punctuation, you must put the complete text in quotes ("").
description (optional)
The description of the process. Some text that describes the process. This text appears in the diagram right above the process symbol. If your text contains punctuation, you must put the complete text in quotes ("").
Example
DiagramName: "create / stop"
DiagramStyle: uml
PageSize: 600,auto
PageMargins: 10,10,10,10

process: p1, Process 1
dummyprocess: p2

create: p1, p2, "create Prozess", "new Prozess", "A newly created process";
stop: p1
