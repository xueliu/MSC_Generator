"\n" - Control Character

A parameter from type text can be broken into multiple lines using the control character "\n".
Syntax Example
command :parameter1, parameter\n2 ;;
Example
DiagramName: Beispiel 2
DiagramStyle: uml
PageSize: 400, auto
PageMargins: 10, 10, 10, 10

process: p1, "Prozess 1"
process: p2, "Prozess 2"

state: p2, Zustand\n2. Zeile\n3. Zeile;
msg: p2, p1, Nachricht\n2-zeilig;
