DIAGRAMSTYLE - Formatting

Use the command diagramstyle to specify the standard to be used when generating the diagram. The parameter style sets the style.
Syntax
diagramstyle: style
Parameters
style
Value Description
uml A UML2.0 conform diagram will be generated.
sdl A SDL conform diagram will be generated.
The default value for this parameter is "uml".
Example
PrintAuthor: yes
PrintCompany: yes
PrintVersion: yes
PrintDate: yes
PrintCreationDate: yes
PrintFileName: yes
Font: "Arial", "10", "Regular"
DiagramName: Call
DiagramStyle: uml
PageMargins: 5, 5, 5, 5
PrintFootLine: yes
