Unified Modeling Language (UML), version 2.1.1

OMG is pleased to announce that the Unified Modeling Language (UML) specification, version 2.1.1, is available.  There are two specifications that comprise the UML 2.1.1 specification: Superstructure and Infrastructure. There are also two specifications that relate to the UML2 specification (Diagram Interchange and Object Constraint Language).  

ISO acceptance:  OMG is pleased to announce that the UML specification (version 1.4.2, OMG document:  formal/05-04-01) is now an accepted ISO specification.  It is available from ISO as 
ISO/IEC 19501.
