"{" - Control Character

Use the curly bracket "{" to define the end of a block, in which all diagram elements will be displayed on a single line ignoring any semi-colons in the block.
Syntax Example
{
Example
DiagramName: Example 5
DiagramStyle: uml
PageSize: 500, auto
PageMargins: 10, 10, 10, 10

process: p1, "Process 1"
process: p2, "Process 2"
process: p3, "Process 3"

state: p1, State 1;
state: p2, State 2;
state: p3, State 3;

{
state: p1, State 4;
state: p2, State 5;
state: p3, State 6;
}
