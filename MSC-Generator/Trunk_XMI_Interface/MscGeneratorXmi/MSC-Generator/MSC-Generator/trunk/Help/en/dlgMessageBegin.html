Dialog MessageBegin

In this dialog you can change the properties of the diagram element MessageBegin. The changes will take effect in the diagram only after you click the button "Apply" or "Apply and Close". The button "Cancel" closes the dialog and you will lose all changes which have not been applied.
Text
The text that should be displayed as message.
See also
msgbegin
Type Asynchronous
The message will be displayed as an asynchronous message.
See also
msgbegin
Type Response
The message will be displayed as a response message.
See also
msgbegin
Type Synchronous
The message will be displayed as a synchronous message. This option is not available in SDL.
See also
msgbegin
Cancel
Closes the properties dialog without applying any changes to the diagram.
Apply
Applies all changes to the diagram.
Apply and Close
Applies all changes to the diagram and closes the properties dialog.
