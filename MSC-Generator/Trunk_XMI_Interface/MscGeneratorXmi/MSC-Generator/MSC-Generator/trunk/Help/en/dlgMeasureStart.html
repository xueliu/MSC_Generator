Dialog MeasureStart

In this dialog you can change the properties of the diagram element MeasureStart. The changes will take effect in the diagram only after you click the button "Apply" or "Apply and Close". The button "Cancel" closes the dialog and you will lose all changes which have not been applied.
Text
The text that will be displayed at the measurement start point.

Cancel
Closes the properties dialog without applying any changes to the diagram.
Apply
Applies all changes to the diagram.
Apply and Close
Applies all changes to the diagram and closes the properties dialog.
