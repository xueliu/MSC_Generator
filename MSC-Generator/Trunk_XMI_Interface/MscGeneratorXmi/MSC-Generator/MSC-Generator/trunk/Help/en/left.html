LEFT - Formatting

The command left can be used to reserve additional place to the left of the diagram. This place can be used for example for comments.
Syntax
left: width
Parameters
width
The width of the additional space. The value is always in pixel.
Example
DiagramName: Comments
DiagramStyle: uml
PageSize: auto
PageMargins: 10,10,10,10
left: 100
right: 50

process: p1, Process 1
process: p2, Process 2

comment: p1, Comment 1, r;
comment: p2, Comment 2, l;
comment: ENV_RIGHT, Comment 3
comment: ENV_LEFT, Comment 4;
linecomment: p2, Comment 5;
linecomment: p1, Comment 6, r, *;
commentoverall: Comment 7;
