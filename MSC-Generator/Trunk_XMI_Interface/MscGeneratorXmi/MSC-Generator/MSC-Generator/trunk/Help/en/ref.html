REF - Diagram Element

Use the command ref to display a ref combined fragment.
Syntax
ref: firstinstanceid, lastinstanceid, text
Parameters
firstinstanceid
The ID of the first lifeline that should be covered by the ref combined fragment.
lastinstanceid
The ID of the last lifeline that should be covered by the ref combined fragment.
text
The text that should appear as the name and parameters of the ref combined fragment. If your text contains punctuation, you must put the complete text in quotes ("").
Example
DiagramName: Combined Fragments
DiagramStyle: uml
PageSize: 600,auto
PageMargins: 10,10,10,10

process: p1, "Process 1"
process: p2, "Process 2"
process: p3, "Process 3"

ref: p1,p2, "Reference 2";
ref: p2,p3, "Reference 3";
