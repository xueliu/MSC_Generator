COMMENT - Diagram Element

Using the command comment diagram comments can be added. The comments can be inserted either at the lifelines of processes or actors, or at the left or right side of the diagram. The position of the comment can be set using the parameters instanceid and orientation The text of the comment is defined by the parameter text. If a comment should be displayed over all instances, then use the command commentoverall instead.
Syntax
comment: instanceid, text, orientation
Parameters
instanceid
The ID of the lifeline at which the comment should be displayed, or one of the following constants:
Value Description
ENV_LEFT lest side of the diagram
ENV_RIGHT right side of the diagram
text
The text that should be displayed as a comment. If your text contains punctuation, you must put the complete text in quotes ("").
orientation (optional)
Sets the position of the comment relative to the lifeline selected by the parameter instanceid.
Value Description
l The comment will be displayed to the left of the lifeline
r The comment will be displayed to the right of the lifeline
This parameter is ignored if the comment will be displayed at the left or right side of the diagram
The default value is for this parameter is "l".
Example
DiagramName: Comments
DiagramStyle: uml
PageSize: auto
PageMargins: 10,10,10,10
left: 100
right: 50

process: p1, Process 1
process: p2, Process 2

comment: p1, Comment 1, r;
comment: p2, Comment 2, l;
comment: ENV_RIGHT, Comment 3
comment: ENV_LEFT, Comment 4;
linecomment: p2, Comment 5;
linecomment: p1, Comment 6, r, *;
commentoverall: Comment 7;
