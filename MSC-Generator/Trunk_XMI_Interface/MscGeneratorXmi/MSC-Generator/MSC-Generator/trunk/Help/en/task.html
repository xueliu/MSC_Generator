TASK - Diagram Element

Use the command task to model SDL specific tasks.
Syntax
task: instanceid, text, orientation
Parameters
instanceid
The ID of the lifeline for which the task should be displayed.
text
The text that should be displayed as the task. If your text contains punctuation, you must put the complete text in quotes ("").
orientation (optional)
Determines the vertical alignment of the state invariant in the diagram line.
Value Description
t The state invariant is top aligned
b The state invariant is bottom aligned
The default value for this parameter is "t".
Example
DiagramName: Tasks
DiagramStyle: sdl
PageSize: auto
PageMargins: 10,10,10,10
left: 100
right: 50

process: p1, Process 1
process: p2, Process 2
process: p3, Process 3

task: p1, Task 1,t
task: p2, Task\n2
task: p3, Task 3,b;
