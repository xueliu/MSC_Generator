Dialog FoundMessage

In this dialog you can change the properties of the diagram element Found Message. The changes will take effect in the diagram only after you click the button "Apply" or "Apply and Close". The button "Cancel" closes the dialog and you will lose all changes which have not been applied.
Message Text
The text that will be displayed in the message.
See also
found
Gate Text
The text that will be displayed under the gate of the found message.
See also
found
Alignment Left
The found message will be displayed to the left of the lifeline.
See also
found
Alignment Right
The found message will be displayed to the right of the lifeline.
See also
found
Cancel
Closes the properties dialog without applying any changes to the diagram.
Apply
Applies all changes to the diagram.
Apply and Close
Applies all changes to the diagram and closes the properties dialog.
