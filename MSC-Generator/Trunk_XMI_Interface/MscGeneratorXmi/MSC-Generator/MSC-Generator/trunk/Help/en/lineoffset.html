LINEOFFSET - Formatting

Using the command lineoffset the height of diagram lines can be set.
Syntax
lineoffset: height
Parameters
height
The height of the space between lines in the diagram. The value of this parameter must be between "1" and "100" (pixel). It is not possible to use other units.
The default value for this parameter is "20".
