.NET Framework 2.0
=================

Bei dem MSC/SD Generator handelt es sich um eine .Net Anwendung, die das Microsoft .NET Framework 2.0 benötigt. Das Microsoft .NET Framework 2.0 ist kostenlos unter folgender Adresse verfügbar:
http://download.microsoft.com/download/5/6/7/567758a3-759e-473e-bf8f-52154438565a/dotnetfx.exe
