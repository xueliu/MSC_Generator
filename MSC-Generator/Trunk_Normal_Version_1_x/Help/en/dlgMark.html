Dialog Mark

In this dialog you can change the properties of the diagram element Mark. The changes will take effect in the diagram only after you click the button "Apply" or "Apply and Close". The button "Cancel" closes the dialog and you will lose all changes which have not been applied.
Comment Text
The text that will be displayed as a comment in the mark.

Cancel

Apply
Applies all changes to the diagram.

Applies all changes to the diagram and closes the properties dialog.
