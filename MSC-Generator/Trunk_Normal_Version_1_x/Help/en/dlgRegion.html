Dialog Region

In this dialog you can change the properties of a lifeline. The changes will take effect in the diagram only after you click the button "Apply" or "Apply and Close". The button "Cancel" closes the dialog and you will lose all changes which have not been applied.
Type Activation
The lifeline is displayed in the activation mode.
See also
regionbegin
Type Suspension
The lifeline will be displayed in suspension mode.
See also
regionbegin
Type Coregion
The lifeline will be displayed in coregion mode.
See also
regionbegin
Cancel
Closes the properties dialog without applying any changes to the diagram.
Apply
Applies all changes to the diagram.
Apply and Close
Applies all changes to the diagram and closes the properties dialog.
