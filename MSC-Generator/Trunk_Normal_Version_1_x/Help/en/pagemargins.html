PAGEMARGINS - Formatting

Use the command pagemargins to set the size of the page margins. The margins are subtracted from the page size which is defined using the command pagesize and therefore the graphical diagram area is reduced. The size of the page margins are specified by the parameters left, top, right, bottom and unit. The parameter unit is optional. In this case the default value "pixel" is used.
Syntax
pagemargins: left, top, right, bottom, unit
Parameters
left
This parameter specifies the width of the left margin. The units used for this value are specified by the optional parameter unit. Decimal numbers should be entered in quotation marks.
The default value for this parameter is "10".
top
This parameter specifies the height of the top margin. The units used for this value are specified by the optional parameter unit. Decimal numbers should be entered in quotation marks.
The default value for this parameter is "10".
right
This parameter specifies the width of the right margin. The units used for this value are specified by the optional parameter unit. Decimal numbers should be entered in quotation marks.
The default value for this parameter is "10".
bottom
This parameter specifies the width of the bottom margin. The units used for this value are specified by the optional parameter unit. Decimal numbers should be entered in quotation marks.
The default value for this parameter is "10".
unit (optional)
This parameter defines the units used for the values of the parameters left, top, right and bottom.
Value Description
pixel Pixel
mm Millimeter
cm Centimeter
inch Inch
The default value for this parameter is "pixel".
