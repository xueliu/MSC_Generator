TEXTCOLOR - Formatting

Use the command textcolor to set the text color of the following text diagram elements. The color is specified by the parameter color and will be used until the next textcolor command changes it.
Syntax
textcolor: color
Parameters
color
The color in which text in diagram elements will be drawn. The color must be specified in the HTML format, for example: "red", "darkgreen", "#FF22A0". If "none" is specified, then the standard value for "white" will be used.
Example
PrintAuthor: yes
PrintCompany: yes
PrintVersion: yes
PrintDate: yes
PrintCreationDate: yes
PrintFileName: yes
Font: "Arial", "10", "Regular"
DiagramName: Colors
DiagramStyle: uml
PageMargins: 5, 5, 5, 5
PrintFootLine: yes
Author: Martin Kotowicz
Company: ITESYS GmbH
Version: 1.0
Date: 20.10.2006
PageSize: auto, auto

process: P1, "Process 1"
process: P2, "Process 2"
process: P3, "Process 3"

backcolor: yellow
fillcolor: #A0A0FF
msg: P1,P2, backcolor
state: P3, fillcolor;

backcolor: none
fillcolor: none
textcolor: red

state: P2, textcolor;
