Dialog Settings

In this dialog you can change the properties of the diagram. The changes will take effect in the diagram only after you click the button "Apply" or "Apply and Close". The button "Cancel" closes the dialog and you will lose all changes which have not been applied.
Show Footer
Configures whether the footer should be displayed when printing or exporting. The height of the footer is subtracted from the height of the worksheet.
See also
printfootline
Author
Sets the author name.

version

Determines whether the diagram's version will be printed in the footer when exporting or printing.

Date
Sets the date that should be printed in the footer.
See also
date
Date Checkbox
Determines whether the diagram's date will be printed in the footer when exporting or printing.

printdate
Today Checkbox
Determines whether the current date will be printed in the footer when exporting or printing.
See also

File Checkbox
Determines whether the diagram's file name will be printed in the footer when exporting or printing.
See also
printfile
Diagram Name
Defines the name of the diagram. The name appears in the top left corner of the diagram. It is not necessary to specify "msc" or "sd".
See also
diagramname
Diagram Style SDL
Create SDL conform diagrams.
See also
diagramstyle
Diagram Style UML2
Create UML 2.0 conform diagrams.
See also
diagramstyle
Export diagram on a single page.
When exporting the diagram will be created on a single page ignoring all manual or automatic page breaks.
Cancel
Closes the properties dialog without applying any changes to the diagram.
Apply
Applies all changes to the diagram.
Apply and Close
Applies all changes to the diagram and closes the properties dialog.
