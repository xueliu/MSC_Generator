Dialog Instance

In this dialog you can change the properties of the diagram element instance (process/actor). The changes will take effect in the diagram only after you click the button "Apply" or "Apply and Close". The button "Cancel" closes the dialog and you will lose all changes which have not been applied.
Process
The name of the instance. The name is only needed for the instance types process and actor. For the type dummy this field is ignored.
See also
process actor dummyprocess
Description
The description of the instance. The description is only used for the types actor and process. This field is ignored for the type dummy.
See also
process actor dummyprocess
Margins Left
Defines additional place to the left of the instance. The value is specified only in pixel.
See also
process actor dummyprocess
Margins Right
Defines additional place to the right of the instance. The value is specified only in pixel.
See also
process actor dummyprocess
Type Process
The instance will be displayed as a process.
See also
process actor dummyprocess
Type Actor
The instance will be displayed as an actor.
See also
process actor dummyprocess
Type Dummy
The instance will not be displayed. A dummy is simply a placeholder for a process that will be created later.
See also
process actor dummyprocess
Cancel
Closes the properties dialog without applying any changes to the diagram.
Apply
Applies all changes to the diagram.
Apply and Close
Applies all changes to the diagram and closes the properties dialog.
