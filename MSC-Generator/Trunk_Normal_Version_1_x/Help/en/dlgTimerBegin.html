Dialog TimerBegin

In this dialog you can change the properties of the diagram element TimerBegin. The changes will take effect in the diagram only after you click the button "Apply" or "Apply and Close". The button "Cancel" closes the dialog and you will lose all changes which have not been applied.
Timer Text
The text that will be displayed at the timer start point.
See also
timerbegin
Alignment Left
The timer will be displayed on the left of the lifeline.
See also
timerbegin
Alignment Right
The timer will be displayed on the right of the lifeline.
See also
timerbegin
Format Standard
The timer will be displayed near the lifeline.
See also
timerbegin
Format Extended
The timer will be displayed far from the lifeline.
See also
timerbegin
Text Inside
The text will be displayed horizontally between the lifeline and the timer line. This option is only available in the extended format.
See also
timerbegin
Text Outside
The text will be displayed horizontally outside of the timer line.
See also
timerbegin
Cancel
Closes the properties dialog without applying any changes to the diagram.
Apply
Applies all changes to the diagram.
Apply and Close
Applies all changes to the diagram and closes the properties dialog.
