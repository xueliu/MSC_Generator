STOP - Diagram Element

Use the command stop to display a stop symbol for a lifeline. The lifeline will not be drawn from this point on.
Syntax
stop: instanceid
Parameters
instanceid
The ID of the lifeline for which a stop symbol should be drawn.
Example
DiagramName: "create / stop"
DiagramStyle: uml
PageSize: 600,auto
PageMargins: 10,10,10,10

process: p1, Process 1
dummyprocess: p2

create: p1, p2, "create Process", "new Process", "A newly created process";
stop: p1
