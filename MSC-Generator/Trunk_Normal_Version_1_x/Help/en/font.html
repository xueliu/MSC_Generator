FONT - Formatting

Use the command font to define the font name and size to be used in the diagram. The parameters fontname, fontsize and fontstyle specify the selected font. The parameters fontsize and fontstyle are optional and if omitted, default values will be used. The specified font will be used for all texts in the diagram.
Syntax
font: fontname, fontsize, fontstyle
Parameters
fontname
The value of this parameter must be the name of a font that is available in the operating system.
The default value for this parameter is "Arial".
fontsize (optional)
This parameter defines the size of the font in pixel. Allowed are values between 6 and 20 pixel. Decimal numbers should be entered in quotation marks.
The default value for this parameter is "10".
fontstyle (optional)
This parameter defines the font style. Multiple style values can be specified but must be separated by spaces.
Value Description
regular Standard
bold Bold
italic Italic
strikeout Stroked out
underline Underlined
Please note that not all fonts support all styles.
The default value for this parameter is "regular".

PrintAuthor: yes
PrintCompany: yes
PrintVersion: yes

PrintFileName: yes
Font: "Arial", "10", "Regular"
